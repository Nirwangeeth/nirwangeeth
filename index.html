Recent Projects

Net Hunter Tool

Net Hunter is a comprehensive network scanning tool built with Python and Nmap. It is designed for ethical hackers and network administrators to perform a range of scans in a unified interface. Net Hunter supports network scanning, port scanning (both minimum and maximum ranges), vulnerability scanning, and exploit scanning. The results are displayed in a user-friendly format with tables for easy interpretation and analysis.

IP Tracker

This IP Tracking Tool provides powerful capabilities for identifying and mapping the geographical locations of IP addresses. Designed for ethical hackers and cybersecurity professionals, this tool enables users to quickly gather crucial information about potential targets, including their geographic region and ISP details. Whether you're conducting penetration tests, investigating suspicious activities, or gathering data for social media intelligence, our IP Tracking Tool is an essential resource for enhancing your cybersecurity efforts.

GUI Port Scanner

GUI Port Scanner is a simple and user-friendly tool that allows you to quickly scan ports on a target network.

GUI Payload Generator

GUI Payload Generator is a simple and user-friendly tool that allows you to quickly generate payloads(msfvenom base payload).
